Data Structures

Description A data structure is a particualr way of organizing data in a computer so that it can be efficiently used. Data structures can implement one or more abstrct data types(ADT),which specify the operations that can be performed on a data structure and the computational complexity of those operations.A data structure is a concrete implementation of the specification provided by the ADT.
